    SOFTENG281: Object Oriented Programming Learn fundamental object-oriented programming concepts using Java, including classes, inheritance, polymorphism, encapsulation, interfaces, and design patterns. The course emphasizes writing clean, modular, and reusable code to solve real-world problems.

    SOFTENG282: Software Engineering Theory SOFTENG282 delves into the theoretical foundations of software engineering, including topics such as formal languages, automata theory, and the design and analysis of algorithms. The course emphasizes logical reasoning and mathematical techniques to understand and solve complex software engineering problems.

    COMPSYS201: Fundamentals of Computer Engineering This course provides an overview of how computers work, from low-level logic to high-level software programming. Students gain introductory skills in designing simple digital systems using Boolean algebra and finite state machines, and implement these designs using Field Programmable Gate Arrays (FPGAs). The course also covers the features and programming of the Microchip ATmega328P microcontroller using the C programming language.

    ENGSCI211: Mathematical Modelling 2 ENGSCI211 is a core course for all engineering students, covering a range of mathematical and modeling topics applicable across all branches of engineering. Topics include ordinary differential equations, multivariable calculus, linear algebra, Fourier series, and introductory data analysis and statistics. The course emphasizes applying mathematical techniques to solve engineering problems.

    SOFTENG284: Data Structures and Algorithms SOFTENG284 introduces essential data structures (such as arrays, linked lists, trees, and graphs) and fundamental algorithms (including sorting, searching, and graph traversal). The course focuses on algorithm efficiency, complexity analysis, and practical implementation, providing students with the tools to solve computational problems effectively.

    SOFTENG283: Software Quality Assurance SOFTENG283 focuses on ensuring software quality through various methodologies and tools. Students learn about testing strategies, debugging techniques, code reviews, and automated testing frameworks. The course emphasizes delivering robust, error-free software by applying quality assurance practices throughout the development lifecycle.

    SOFTENG206: Software Engineering Design 1 SOFTENG206 introduces students to systematic software development practices. The course covers skills and tools for software design, including testing, version control, and build systems. Emphasis is placed on collaborative development, ensuring students are well-prepared for real-world software engineering projects.

    ENGGEN204: Professional Skills, Communication, and Collaboration ENGGEN204 focuses on developing essential professional skills, with a strong emphasis on effective communication and collaboration. Students engage in group work, learning about teamwork, problem analysis, and presentation skills. The course aims to equip students with the interpersonal skills necessary for success in engineering and professional environments.
